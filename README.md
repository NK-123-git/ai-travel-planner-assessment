# AI Travel Planner Assessment

## Project Overview

AI Travel Planner is a multi-user travel planning platform that generates personalized travel itineraries using Large Language Models (LLMs).

Users can:

- Register and Login securely
- Create travel plans
- Generate AI-powered itineraries
- Estimate travel budgets
- Modify travel schedules
- Receive hotel recommendations
- Generate smart packing checklists

---

## Core Features

### Authentication & Authorization
- Secure user registration
- JWT-based authentication
- Protected routes
- User data isolation

### AI Itinerary Generation
Generate day-by-day travel schedules based on:
- Destination
- Duration
- Budget
- Interests

### Budget Estimation
Generate estimated travel expenses:
- Flights
- Accommodation
- Food
- Activities

### Editable Itinerary
Users can:
- Add activities
- Remove activities
- Regenerate specific days

### Hotel Suggestions
AI-generated hotel recommendations based on:
- Destination
- Budget preference
- Popular traveler ratings

---

## Custom Feature

### AI Weather-Aware Packing Assistant

The system generates a personalized packing checklist based on:
- Destination
- Weather conditions
- Travel duration
- Planned activities
